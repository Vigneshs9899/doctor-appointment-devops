# Doctor Appointment App (DevOps Project)

## Features
- Book appointment via UI
- Backend API (Flask)
- PostgreSQL database (Azure)
- Dockerized application
- CI/CD pipeline (GitHub Actions)
- Deployed on Azure Container Apps

## Architecture
Frontend → Backend → Database → Cloud

## Tech Stack
- Python (Flask)
- Docker
- GitHub Actions
- Azure Container Apps
- Azure PostgreSQL

## How it works
Push → Build → Push Image → Deploy → Live App
