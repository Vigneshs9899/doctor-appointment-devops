# 🌿 Shayure Ayurveda - Doctor Appointment Management System

A modern full-stack Doctor Appointment Management System built using Flask, PostgreSQL and Docker. The application allows patients to book appointments online while providing doctors with a dashboard to manage consultations, generate medical reports and monitor appointment statistics.

---

🌐 Live Website:
https://doctor-appointment-devops.onrender.com/
📂 GitHub Repository:
https://github.com/Vigneshs9899/doctor-appointment-devops.git

## ✨ Features

### 👤 Patient
- User Registration & Login
- Book Doctor Appointments
- Search Appointments
- View Appointment History
- Download Medical Report (PDF)

### 👨‍⚕️ Doctor
- Secure Doctor Login
- Dashboard Analytics
- Manage Consultations
- Update Appointment Status
- Add Diagnosis
- Add Prescription
- Add Doctor Notes
- Schedule Follow-up
- Generate Professional PDF Reports

### 📊 Dashboard
- Appointment Statistics
- Monthly Appointment Charts
- Appointment Status Distribution
- Responsive Dashboard

### 🌐 Landing Page
- Modern Hero Section
- Responsive Design
- About Section
- Services
- Contact Section

## ⚙ Installation

Clone repository

```bash
git clone https://github.com/Vigneshs9899/doctor-appointment-devops.git
```

Install packages

```bash
pip install -r requirements.txt
```

Run application

```bash
python app.py
```

---

## Docker

```bash
docker-compose up --build
```
